Personal Projects



Mastermind

Mastermind is two player game where one player is the codemaker and the other player is the codebreaker. The codebreaker is given ten tries to guess the code created by the codemaker. A one and two player version of this game was created in MATLAB using simple game engine. In this version, the codebreaker enters their guess by typing the first letter of each color in the same line. The corresponding colored pegs, along with white and black pegs that represent right and wrong spots, are displayed on the game board. After the game ends, the code is revealed and the player/players are asked if they want to play again.
